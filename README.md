# UXCircuit Landing Page

Single-file landing page. No build tools, no dependencies.

## Files
- `landing.html` — the entire site (HTML + CSS + JS, all inline)

## How to host on GitHub Pages

1. Create a new repo on GitHub (e.g. `uxcircuit-landing`)
2. Upload `landing.html` to the repo
3. Rename it to `index.html` (GitHub Pages serves `index.html` by default)
4. Go to repo Settings → Pages → Source → Deploy from branch → main → / (root) → Save
5. Your site will be live at: `https://YOUR-USERNAME.github.io/uxcircuit-landing`

GitHub Pages gives you a free `https://` domain — YouTube embeds will work immediately.

## Notes
- YouTube videos require `https://` to embed. They will NOT play from `file://` (local).
- Tally.so waitlist popup is already wired up and will work on any domain.
- The logo marquee, scroll animations, and lightbox all work without any server.
